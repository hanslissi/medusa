# Medusa Cache In-memory

Medusa in-memory cache module. Use plain JS Map as a cache store.

## Installation

```
yarn add @medusajs/cache-inmemory
```

## Options

```
{
  ttl?: number // Time to keep data in cache (in seconds)
}
```
